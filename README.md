Data Interchange Format Implementations

- Comma-Separated Values

## Build

```shell
git clone

```

```shell
echo 'export PATH="$PATH:/usr/local/bin/htdi"' >> ~/.bashrc
sudo echo 'export PATH="$PATH:/usr/local/bin/htdi"' >> /etc/skel/.bashrc
```
