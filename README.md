- Comma-Separated Values

- [JavaScript Object Notation][JSON]
- [Markdown][CommonMark]
- [Tom's Obvious Minimal Language][TOML]
- Web Socket Protocol
- [Web Transport Protocol][Web Transport]
- [YAML Ain't Markup Language][YAML]
